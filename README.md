## Running locally

Install Node.js if you don't have it. [Download Node.js](https://nodejs.org/)

Install yarn if you don't have it:

```powershell
npm i -g yarn
```

Install dependencies:

```powershell
yarn
```

Run:

```powershell
yarn dev
```

## Building

Install yarn if you don't have it:

```powershell
npm i -g yarn
```

Install dependencies:

```powershell
yarn
```

Build:

```powershell
yarn build
```
